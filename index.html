game engine

A simple but complete, game engine build with HTML5 and Javascript. There are classes to control the game loop, spritesheet animations, keyboard interaction and collision between game objects.

About the game:

Take control of your spaceship and beat up all Ovni's in the space! Don't crash with the enemys and try to make the highest score that you can. Use the arrows to move, space to shoot and enter to pause the game.

Developed by Caue Queiroz.
